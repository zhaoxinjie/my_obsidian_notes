# 大语言模型智能体（LLM-based Agents）

## 定义

`LLM-based Agents` 指以大语言模型作为核心决策与语言能力中枢，并结合感知、记忆、规划、工具调用或环境交互能力来完成任务的智能体系统。

## 为什么重要

它把 LLM 从“回答问题的模型”推进到“可感知、可决策、可执行”的系统角色，是通向更通用 AI 系统的一条重要路径。

## 当前理解

- LLM 往往承担 brain 角色，但完整 agent 还需要 perception 与 action。
- agent 的价值不只在回答，更在持续任务执行、规划和环境交互。
- 研究与应用通常分成单智能体、多智能体、人机协作三类。
- 这个领域同时牵涉结构设计、评测、训练、可靠性和社会行为。
